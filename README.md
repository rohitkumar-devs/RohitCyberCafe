# Rohit Cyber Cafe Website

A frontend website built for a real-world local cyber café business to showcase services and provide basic online presence.

## Purpose
This project was created to represent a local cyber cafe that provides digital services such as online forms, government schemes, and basic IT support to villagers.

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Features
- Service listing for cyber café offerings  
- Simple and user-friendly layout  
- Designed for non-technical users  

## What I Learned
- Building websites for real users, not just practice  
- Structuring content for clarity and accessibility  
- Applying frontend fundamentals in a real business scenario  

## Notes
This project reflects practical exposure to handling real customer-facing technology in a non-urban environment.
